# ImmigrationAnalysis

In this project we tried to analyze different trends and patters that occur during the process of migration.

## Dataset and Insiprations: 

1. [2013 American Community Survey | Kaggle](https://www.kaggle.com/census/2013-american-community-survey) 
2. [International Database Of Migration](https://stats.oecd.org/Index.aspx?DataSetCode=MIG) 
3. [Data.gov](https://catalog.data.gov/dataset?vocab_category_all=Immigration)
4. [Infographics 2015 | Homeland Security](https://www.dhs.gov/immigration-statistics/visualization/2015)
5. [Mapping Immigration | Homeland Security](https://www.dhs.gov/mapping-immigration)
